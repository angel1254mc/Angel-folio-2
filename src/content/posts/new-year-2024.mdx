### Internship at Grafana Labs (June 2023 - September 2023)

Last Summer, I had the wonderful opportunity to work at Grafana Labs Inc. as a Software Engineer Intern in the cloud-onboarding team! Here, I got to work on an integration for Grafana's Cloud Platform that allows users to import metrics from any publicly-addressable metrics endpoint (In prometheus format, typically the URL will look something like "https://my-url.com/metrics"). It works by reading the URL(s) and authentication details provided by a user, then scraping that metrics endpoint using one (or more!) scrapers hosted in Grafana Cloud's infrastructure. It was a really cool project to work on, and I'm really happy that the integration is available to Grafana Cloud users on all tiers!
